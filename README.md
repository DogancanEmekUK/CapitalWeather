#CAPITAL WEATHER APP

-This app allows you to see the weather conditions of European capitals in a detailed way.
-It's both educational and great for daily use.
-The data is from: https://www.weatherapi.com

#USED TECHNOLOGIES

-MVVM
-DataBinding
-ViewBinding
-Coroutines
-Retrofit
-Navigation
-LiveData
-LifeCycle
-Glide
